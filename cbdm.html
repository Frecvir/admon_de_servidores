CONFIGURACIÓN DE UNA BASE DE DATOS MYSQL EN UN SERVIDOR WEB

La configuración de una base de datos MySQL en un servidor web implica instalar, inicializar y optimizar el entorno necesario para gestionar bases de datos relacionales que permitan almacenar y recuperar información de manera eficiente para aplicaciones web. MySQL es uno de los sistemas de gestión de bases de datos (DBMS) más utilizados en el desarrollo web debido a su velocidad, fiabilidad y compatibilidad con múltiples plataformas.


PASOS PARA CONFIGURAR
  1. Instalar MySQL en el Servidor
    • En sistemas Linux (Ubuntu):
      • sudo apt update
sudo apt install mysql-server
    • En sistemas Windows: Descarga el instalador desde el sitio oficial de MySQL.
    • Inicia el servicio MySQL:
      • sudo service mysql start
  2. Configurar la Base de Datos
    • Inicia la consola MySQL:
      • sudo mysql
    • Cambia la contraseña del usuario root:
      • ALTER USER 'root'@'localhost' IDENTIFIED WITH 
				
      mysql_native_password BY 'contraseña_segura';
FLUSH PRIVILEGES;
    • Crea una nueva base de datos:
      • CREATE DATABASE mi_base_datos;
  3. Configurar el Acceso Remoto
    • Edita el archivo de configuración:
      • sudo nano /etc/mysql/mysql.conf.d/mysqld.cnf
    • Comenta la línea:
      • #bind-address = 127.0.0.1
    • Reinicia MySQL para aplicar los cambios:
      • sudo service mysql restart
  4. Crear un Usuario y Otorgar Permisos
    • Crear un usuario con acceso remoto:
      • CREATE USER 'usuario'@'%' IDENTIFIED BY 'contraseña';
GRANT ALL PRIVILEGES ON mi_base_datos.* TO 'usuario'@'%';
FLUSH PRIVILEGES;
